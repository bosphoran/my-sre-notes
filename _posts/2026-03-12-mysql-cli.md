## MySQL CLI Commands

### 🔑 User Management

```sql
SELECT user, host FROM mysql.user; -- Show all users
CREATE USER 'username'@'%' IDENTIFIED BY 'password'; -- Create a new user
GRANT ALL PRIVILEGES ON dbname.* TO 'username'@'%'; -- Grant privileges
FLUSH PRIVILEGES;
ALTER USER 'username'@'%' IDENTIFIED BY 'newpassword'; -- Change user password
DROP USER 'username'@'%'; -- Drop a user
```

### 📂 Database Management

```sql
SHOW DATABASES; -- Show all databases
CREATE DATABASE mydb; -- Create a new database
USE mydb; -- Use a database
DROP DATABASE mydb; -- Drop a database
```

### 📋 Table Management

```sql
SHOW TABLES; -- Show all tables in current database
CREATE TABLE users (
id INT AUTO_INCREMENT PRIMARY KEY,
name VARCHAR(50),
email VARCHAR(100)
); -- Create a table
DESCRIBE users; -- Describe table structure
DROP TABLE users; -- Drop a table
```

### ✏️ Data Manipulation

```sql
INSERT INTO users (name, email) VALUES ('Alice', 'alice@example.com'); -- Insert data
SELECT * FROM users; -- View data
UPDATE users SET email='alice@newdomain.com' WHERE name='Alice'; -- Update data
DELETE FROM users WHERE name='Alice'; -- Delete data
```

### 🛠 Useful Admin Command

```sql
SELECT USER(); -- Show current user
SELECT DATABASE(); -- Show current database
EXIT; -- Exit MySQL CLI
```

Working with MySQL inside a K8s pod.

```bash
kubectl exec -it <mysql-pod-name> -- mysql -u root -p # Connect as root inside the MySQL pod
mysql -h mysql-service -u zafar -p # login into mysql using user zafar
```
